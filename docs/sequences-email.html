⚙️ Setup Brevo — Étapes à faire 1 seule fois

  1. 1Crée un compte Brevo gratuit sur brevo.com (plan gratuit = 300 emails/jour, automation incluse)
  2. 2Crée 4 listes de contacts :
    Lead Magnet (ID 2) — Acheteurs Formation (ID 3) — Acheteurs TikTok Shop (ID 4) — Acheteurs Accompagnement (ID 5)
    Note les IDs exacts affichés dans Brevo — ils peuvent être différents
  3. 3Récupère ta clé API Brevo : Settings → API keys → Generate a new API key
  4. 4Ajoute ces variables d'environnement dans Netlify : Site → Site configuration → Environment variables
    BREVO_API_KEY=ta-cle-api-brevo-ici
    BREVO_LEADMAGNET_LIST=2
    BREVO_FORMATION_LIST=3
    BREVO_SHOP_LIST=4
    BREVO_ACCOMPAGNEMENT_LIST=5
  5. 5Crée les automations Brevo : Automation → Create a workflow → "Contact added to a list" comme trigger.
    Une automation par liste = une séquence. Copie-colle les emails ci-dessous.
  6. 6Configure ton expéditeur : Settings → Senders & IPs → Add a sender : VyraShop Academy / contact@vyrashop.com + valide le domaine
  7. 7Mets le PDF de l'ebook en ligne : push ebook_tiktok_performance.pdf à la racine de ton repo GitHub. L'URL sera https://vyrashop.com/ebook_tiktok_performance.pdf
